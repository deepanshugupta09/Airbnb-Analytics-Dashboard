## Overview
This project is a comprehensive Airbnb Analytics Dashboard developed using Power BI. The dashboard provides deep insights into Airbnb listings, customer reviews, booking trends, pricing patterns, host performance, and occupancy analysis.

The project focuses on transforming raw Airbnb data into meaningful business insights using interactive visualizations and advanced DAX calculations.

---

## Project Objectives
- Analyze Airbnb listing performance
- Identify pricing and occupancy trends
- Evaluate customer review behavior
- Monitor host performance metrics
- Create interactive dashboards for business insights
- Improve data-driven decision-making
